# qqmusic lyrics finder
## やり方
リポジトリをフォーク  
↓  
Actions画面に行く。  
↓  
Manual lyric generationを選択する  
↓  
Run workflowからいろいろ引数を指定して実行してください(Actionsの使用量に注意してください)  
## 引数解説
* Video Id:YoutubeのビデオのIdを入れてください  
* Number of lines to skip (QRC)と(LRC):これは最初の何行かを無視するオプションです。  
qqmusicの歌詞の先頭にはメタ情報が書かれており、手動で無視することができます。
